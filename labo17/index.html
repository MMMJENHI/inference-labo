🎓 Étape 3 : L'Équation aux Différences

C'est ici que l'on transforme les coefficients en un signal filtré. Le filtre « Biquad » traite le signal échantillon par échantillon ($n$).

// Algorithme Direct Form I
y[n] = (b0 * x[n]) + (b1 * x[n-1]) + (b2 * x[n-2])
- (a1 * y[n-1]) - (a2 * y[n-2])

Partie Non-Recursive (Zéros)

Utilise les entrées actuelles et passées ($x$). C'est la moyenne glissante pondérée.

Partie Recursive (Pôles)

Réutilise les sorties passées ($y$). C'est ce qui crée la « mémoire » et la raideur du filtre.

Implémentation C++ pour Microcontrôleur

float x1=0, x2=0, y1=0, y2=0; // Mémoires float applyFilter(float input) { // Coefficients calculés float b0 = 0.067339f; float b1 = 0.134678f; float b2 = 0.067339f; float a1 = -1.143742f; float a2 = 0.413098f; // Équation aux différences float output = b0*input + b1*x1 + b2*x2 - a1*y1 - a2*y2; // Mise à jour des états (delay line) x2 = x1; x1 = input; y2 = y1; y1 = output; return output; }
